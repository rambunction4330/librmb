Detailed Description

Odometry for a differential drive based on encoder and gyroscope data. It can also optionaly incorperate vision based odometry over NetworkTables.

Constructs a DiffernetialDriveOdometry object using encoder, gyro and vision data.

Parameters
leftEncoderA shared pointer to the left side encoder. If there are multiple, the back wheel is usually prefered.
rightEncoderA shared pointer to the right side encoder. If there are multiple, the back wheel is usually prefered.
trackWidthThe distance between the left and right side wheels for kinematics.
gyroA shared pointer to a gyroscopefor determining the robots heading.
visionTableA shared pointer to the NetworkTable holding a vision based estimation of robot position. This table must include a DoubleArrayTopic named "pose". The array must have three elements ordered X, Y, Heading. X and Y must be in meters and heading must be in radians. The table may also include a DoubleArrayTopic named "stdDev" as how much this estimation can be trusted. The array is also ordered X, Y, Heading with X and Y in meters and Heading in radians. When using the "stdDev" topic, always update the standard deviations befor the position.
initalPoseInital position of the robot.
